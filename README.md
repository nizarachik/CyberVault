# CyberVault
CyberVault est une plateforme de cybersécurité tout-en-un offrant un coffre-fort chiffré AES-256, un générateur et gestionnaire de mots de passe, des QR Codes sécurisés, une messagerie chiffrée, un coffre de données sensibles, un simulateur de robustesse, un analyseur de fichiers et un conseiller IA, le tout avec une interface moderne.
